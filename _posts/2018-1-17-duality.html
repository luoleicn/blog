拉格朗日对偶引出

minimizef0(x)s.t.fi(x)⩽0,i=1,...,mhi(x)=0,i=1,...,p \begin{matrix} minimize \quad f_0(x) \quad \\ s.t. \quad f_i(x) \leqslant 0, \quad i=1,...,m \\ \quad \quad h_i(x) = 0, \quad i=1,...,p \end{matrix}minimizef0​(x)s.t.fi​(x)⩽0,i=1,...,mhi​(x)=0,i=1,...,p​

理解拉格朗日函数

重新将原问题描述为无约束问题：

minimizef0(x)+∑i=1mI−(fi(x))+∑i=1pI0(hi(x))minimize \quad f_0(x) + \sum_{i=1}^m I_{-} (f_i(x)) + \sum_{i=1}^p I_0(h_i(x))minimizef0​(x)+i=1∑m​I−​(fi​(x))+i=1∑p​I0​(hi​(x))

其中：

I−(u)={0,  u⩽0∞,  u>0I0(u)={0,  u=0∞,  u≠0\begin{matrix} I_-(u) = \left\{ \begin{array}{lll} 0, \ \ u \leqslant 0 \\ \infty, \ \ u>0 \end{array} \right. \\ \\ I_0(u) = \left\{ \begin{array}{lll} 0, \ \ u = 0 \\ \infty, \ \ u \neq 0 \end{array} \right. \end{matrix}I−​(u)={0,  u⩽0∞,  u>0​I0​(u)={0,  u=0∞,  u≠0​​

I−I_-I−​和I0I_0I0​都是示性函数，可以理解为我们对约束条件的不满意程度，当约束条件满足时，不满意程度为0，当约束条件被打破时，不满意程度为无穷大。

L(x,λ,υ)=f0(x)+∑i=1mλifi(x)+∑i=1pυihi(x)s.t.λi⩾0,i=1,...mυi⩾0,i=1,...p\begin{matrix} L(x, \lambda, \upsilon) = f_0(x) + \sum_{i=1}^m \lambda_i f_i(x) + \sum_{i=1}^p \upsilon_ih_i(x) \\ s.t. \quad \lambda_i \geqslant 0, \quad i=1,...m \\ \quad \quad \upsilon_i \geqslant 0, \quad i=1,...p \end{matrix}L(x,λ,υ)=f0​(x)+∑i=1m​λi​fi​(x)+∑i=1p​υi​hi​(x)s.t.λi​⩾0,i=1,...mυi​⩾0,i=1,...p​

在拉格朗日函数的表达中，使用λ\lambdaλ和υ\upsilonυ代替里I−I_-I−​和I0I_0I0​，λ\lambdaλ和υ\upsilonυ都是大于零的整数，可以理解为不满意函数为柔性的，不再是无穷大，而是随着越来越违背约束条件，就越来越不满意。

示例一：线性函数对偶推导

线性规划的一般形式：

L(x)=cTx+λT(Ax−b)s.t.λi⩾0x⩾0\begin{matrix} L(x) = c^Tx + \lambda^T(Ax - b) \\ s.t. \quad \lambda_i \geqslant 0 \\ \quad \quad x \geqslant 0 \end{matrix}L(x)=cTx+λT(Ax−b)s.t.λi​⩾0x⩾0​

1）当cT+λTA<0c^T + \lambda^TA < 0cT+λTA<0时，g(λ)=minimizex  (cT+λTA)x−λTb=−∞g(\lambda) = minimize_x\ \ (c^T + \lambda^TA)x - \lambda^Tb = -\inftyg(λ)=minimizex​  (cT+λTA)x−λTb=−∞对偶不可行，无意义

为什么要解对偶问题而不死后原问题本身

  • 当原问题不知道如何求解的时候，解对偶问题至少可以得到一个原问题的下界

  • 有些问题的原问题很难求解或者无法求解，而对偶问题缺可以比较方便的求解，如上面示例的支持向量机优化对偶问题催生了SMO方法，这个方法要比直接去优化原问题的二次规划问题方便的多

  • 对偶问题sensitivity analysis会更容易。如果对原问题的约束条件进行变化，很可能之前求的解就不再可行范围里了，但是对于对偶问题，只是优化目标发生了变化，原来的解还是可行的（很可能距离新的最优解也不远）

  • 原问题的最小值可能根本无法求得，通过对偶问题可以发现这一点。
